python自动聊天机器人

2020-04-26

你知道如何利用python玩转微信吗，你是否因为工作忙没时间陪女朋友? 今天给大家介绍一个利用微信个人号API库wxpy实现微信聊天机器人，只需要寥寥数行代码，它就能够自动陪朋友聊天，消息秒回。而且无所不知，甚至还能问天气，讲笑话。 从此妈妈再也不用担心我会错过微信消息了。

wxpy介绍

wxpy是一个微信机器人库，在 itchat 的基础上，通过大量接口优化提升了模块的易用性，并进行丰富的功能扩展。 支持一些常见的场景:

  • 控制路由器、智能家居等具有开放接口的玩意儿
  • 运行脚本时自动把日志发送到你的微信
  • 加群主为好友，自动拉进群中
  • 跨号或跨群转发消息
  • 自动陪人聊天
  • 逗人玩
  • …

总之，利用wxpy我们可以实现微信个人号的各种自动化操作。

实现思路

在这里我们再介绍一个人工智能机器人微软小冰，它是一个领先的跨平台人工智能机器人。在微信搜索人工智能小冰微信公众号，就可以找到并使用它。那么我们的思路就是：利用wxpy登录微信，然后将朋友发送回来的消息发送给小冰，然后再将小冰的消息发送给朋友。说白了，就是将自己的微信作为一个中间人，利用小冰当自己的私人助理。

登录python官网，找到适合你操作系统的python安装包，并下载安装。推荐3.4以上版本。

由于本文主要不是讲解python环境的搭建，再次不详细叙述，大家可以自行搜索教程。再次假定你已经装好并配置好了环境变量。

安装wxpy库

打开cmd或者powershell. 使用pip命令安装wxpy.

安装好之后，开始编写代码。
